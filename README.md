# Beyond Canvas

Beyond Canvas is an open-source framework to speed up the development of apps for ePages Beyond Shops.

## Requirements

Rails 7
Ruby >3.1

## Installation
Add this line to your application's Gemfile:

```ruby
gem "beyond_canvas"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install beyond_canvas
```
